# Nursultan account checker | selenium
![GitHub top language](https://img.shields.io/github/languages/top/OkulusDev/Oxygen)

## **Nursultan Checker - однопоточный чекер аккаунтов для сайта nursultan.fun​**
<hr style="border:2px solid gray">

# Главные фичи

- Обход Cloudflare Turnstile с помощью seleniumbase (автоматически наводится)
- valid.txt - валидные акки с ником, email и статусом подписки.
- invalid.txt - невалидные (с причиной).
- errors.txt - ошибки (нет токена, Cloudflare, таймауты).
- Сбор информации - в случае валидного аккаунта, собираются такие данные как: email,наличие подписки на аккаунте(так же оставшиеся время до окончание подписки)
<hr style="border:2px solid gray">

# Ошибка
может возникнуть длинная ошибка 403 в терминале "проверяем человек ли вы" это капча Cloudflare Turnstile Challenge Page(вроде) ебет мозги, нужно перезагрузить сайт и пройти капчу и все заработает дальше
